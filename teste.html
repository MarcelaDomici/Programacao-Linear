Sobre o Projeto

Uma plataforma para análise de algoritmos de otimização

Nossa Missão

Este projeto foi desenvolvido para explorar e comparar diferentes abordagens de algoritmos de otimização. Através de uma interface intuitiva, é possível selecionar entre algoritmos como Subida de Encosta, Subida de Encosta com Tentativas e Tempera Simulada, ajustando parâmetros como tipo de execução e tamanho do problema.

Configuração Flexível
